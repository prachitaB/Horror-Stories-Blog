# 1. Project Overview

## Introduction

Horror Stories Blog is a static web application designed to provide an immersive platform for exploring horror stories, paranormal phenomena, urban legends, haunted places, supernatural beings, and unexplained mysteries.

The website combines the functionality of a horror blog with a digital encyclopedia, allowing users to browse categorized information through an intuitive and visually engaging interface.

The project has been developed as an academic web development project while following modern web design principles and responsive design practices.

---

# 2. Objectives

The primary objectives of the project are to:

* Create an engaging horror-themed website.
* Organize paranormal information into structured categories.
* Provide an easy-to-navigate user interface.
* Showcase responsive web design techniques.
* Demonstrate the use of HTML, CSS, and JavaScript.
* Deliver an informative and visually appealing browsing experience.

