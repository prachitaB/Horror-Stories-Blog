# Horror Stories Blog

## Project Report

---

# Abstract

Horror Stories Blog is a static web application designed to provide users with an immersive horror-themed experience through organized paranormal archives, horror stories, urban legends, haunted places, and unexplained phenomena.

---

# Objectives

- Learn HTML
- Learn CSS
- Learn JavaScript
- Build a responsive website
- Organize horror information

---

# Scope

The website serves as:

- Horror Encyclopedia

- Paranormal Archive

- Story Collection

- Educational Resource
